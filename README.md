# Hotel Reservation Database

Este projeto consiste na modelagem e implementação de um banco de dados para um sistema de reservas de hotel, desenvolvido utilizando SQL e MySQL.

## 📌 Sobre o projeto

O banco de dados foi projetado para gerenciar informações de um hotel, incluindo clientes, quartos, reservas e pagamentos. O sistema permite registrar reservas, consultar disponibilidade de quartos e armazenar dados dos hóspedes.

## 🗄️ Estrutura do Banco de Dados

O banco de dados é composto por tabelas principais, como:

* Clientes
* Quartos
* Reservas
* Pagamentos
* Funcionários (opcional)

## ⚙️ Funcionalidades

* Cadastro de clientes
* Cadastro de quartos
* Registro de reservas
* Consulta de disponibilidade
* Atualização de dados
* Exclusão de registros
* Consultas SQL com JOIN
* Consultas com GROUP BY
* Relatórios de reservas

## 🛠️ Tecnologias utilizadas

* MySQL
* SQL
* MySQL Workbench
* Modelagem ER (Diagrama Entidade-Relacionamento)

## 📁 Arquivos do projeto

* Script de criação das tabelas
* Script de inserção de dados
* Script de consultas SQL
* Script de atualização e exclusão
* Diagrama ER

## ▶️ Como executar o projeto

1. Abrir o MySQL Workbench.
2. Criar um novo banco de dados.
3. Executar o script `create.sql` para criar as tabelas.
4. Executar o script `insert.sql` para inserir dados.
5. Executar os scripts de consultas para testar o banco.
